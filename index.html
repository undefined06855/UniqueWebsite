Well, how does this work?

Magic. Or just using your IP address.

This website uses ipify.org to get your IP address (which is obviously going to be unique), then converts it to a number, and plugs it into a hsl calculation to get a unique background colour for you!
The number used is the same as the one you see above.

The main code to generate that number was created by chatGPT, and you can see it below: 

                    
function uniqueValue(arr)
{
    let unique = 0
    for (let i = 0; i < arr.length; i++) {
        unique += arr[i] * Math.pow(256, i)
    }
    return unique
}

function uniqueValue2(arr)
{
    const unique = uniqueValue(arr);
    const mapped = unique / (Math.pow(256, 4) - 1);
    return mapped;
}

uniqueValue takes in a list of four (or more) values, which are the four digits of your IP address. uniqueValue2 just maps it to a 0 - 1 range, so you can use it with css's turn to get an angle value for hsl, and create the background colour!

There's a bit more code, like storing the IP in localStorage so you don't have to ping ipify every time, but that's not important. If you really want to know, just look yourself.
